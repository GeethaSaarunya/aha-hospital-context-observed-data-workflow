# Reproducibility Notes

This repository is a **code-and-metadata release**, not an open data release.

## Included
- Scripts 13–17 used for the reported workflow
- Aggregate outputs that are non-disclosive
- Figures and manuscript summary tables
- Templates and metadata needed to understand the workflow

## Excluded
- Raw AHA data
- Row-level derived hospital datasets
- Hospital-level profile assignment files
- System-level tables containing identifiable system-group codes
- Local path configuration RDS files

## Public versus restricted outputs
Some files generated during the local analysis environment are intentionally omitted because they depend on restricted AHA inputs and may contain row-level or system-identifiable information. The public package is limited to materials appropriate for code review, workflow inspection, manuscript audit, and re-use by authorized data holders.
