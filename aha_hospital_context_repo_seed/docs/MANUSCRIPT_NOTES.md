# Manuscript note

The public repository is intended to support the manuscript workflow and figures.
The manuscript document itself is not required for code reuse and can be versioned separately if desired.

If you later decide to include the manuscript in the GitHub repository, consider storing:
- the current DOCX or Markdown source
- figure captions
- the submission cover letter
- journal-specific checklists

These are intentionally kept out of the current seed package to keep the public archive focused on code, metadata, and non-disclosive outputs.
