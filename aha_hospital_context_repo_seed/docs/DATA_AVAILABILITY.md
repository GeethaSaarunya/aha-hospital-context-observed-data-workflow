# Data Availability

The underlying 2023 American Hospital Association (AHA) Annual Survey Database and AHA Information Technology dataset used in this project are proprietary and licensed and therefore are **not redistributed** in this repository.

This repository provides:

- analytic scripts used for the paper
- a variable manifest
- a file-layout crosswalk
- mock input templates
- non-disclosive aggregate outputs used in the manuscript
- manuscript figures and summary tables
- repository metadata files for GitHub + Zenodo release

Users who independently obtain authorized access to the AHA source files may adapt the workflow locally. Raw data files, row-level merged data, hospital-level profile assignments, and system-identifiable outputs are intentionally excluded from this public package.
