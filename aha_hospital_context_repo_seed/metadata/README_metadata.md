# Metadata files

- `variable_manifest.csv` documents the analysis variables considered in the public workflow and their filtering status.
- `file_layout_crosswalk.csv` maps logical workflow inputs to expected local restricted files.
- `selected_asdb_profile_vars_template.csv` and `selected_it_profile_vars_template.csv` are templates for selected-variable lists expected by the scripts.

These files are intended to make the workflow auditable without redistributing the licensed source data.
