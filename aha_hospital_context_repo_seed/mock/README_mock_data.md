# Mock templates

These files are empty templates that document the minimum column structure expected by the public workflow materials.

- `aha_asdb_2023_template.csv` documents example columns from the licensed AHA ASDB input.
- `aha_it_2023_template.csv` documents example columns from the licensed AHA IT input.

They are not real data and cannot reproduce the study by themselves. Authorized users must independently obtain access to the licensed AHA source files and map their local file names and layouts to the workflow using `metadata/file_layout_crosswalk.csv`.
